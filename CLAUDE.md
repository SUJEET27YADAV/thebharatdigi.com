# localhost Design System

This project uses the **localhost** design system extracted by skillui.

## How to use

Read `SKILL.md` in the directory `.agents/skills/design/` for the full design system reference before writing any UI code.

Key files:

- `.agents/skills/design/SKILL.md` — master design reference (read this first)
- `.agents/skills/design/references/DESIGN.md` — extended tokens and component specs
- `.agents/skills/design/references/ANIMATIONS.md` — motion and keyframe specs
- `.agents/skills/design/references/LAYOUT.md` — grid and layout containers
- `.agents/skills/design/references/COMPONENTS.md` — DOM component patterns
- `.agents/skills/design/screens/scroll/` — scroll journey screenshots (study before implementing)

When building any UI, always read SKILL.md first and match colors, fonts, spacing, and motion exactly.
